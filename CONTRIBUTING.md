# Contributing to ZT-IGB
We welcome contributions from the U.S. cybersecurity community, federal IT contractors, and Zero Trust practitioners.

## How to Contribute
1. **Report Vulnerabilities:** If you find a flaw in the compliance templates, please see `SECURITY.md`.
2. **Submit Blueprints:** We encourage the submission of new Terraform or CloudFormation templates that map to NIST 800-53 controls.
3. **Documentation:** Help us improve the "National Security Mapping" of these technical controls.

By contributing, you help harden the digital infrastructure of the United States.
